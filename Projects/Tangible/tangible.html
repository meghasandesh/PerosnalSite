Tangible Anchring

Tangible interaction (TUIO) allows us to use real world objects to interact computer programs, providing a novel and relatable paradigm for interaction and cmmunication. The tangible anchroing system brings the TUIO into the field of visualization and combines it with the tabletop computer.

The result is a dynamic program which stimulates cooperative discovery, debate and discussion with rich interactions.

We have used the system to presnt data gathered political opinion and videos gathered using a mobile app.

AquaMobile

AquaMobile is a mobile-web application for retrieving and recording data about fisheries.

The application consists of the following interfaces:

Google Maps Interface with KML Overlay: Fisheriy boundaries are displayed as KML overlays on the Google Maps interface. Clicking on the region opens a popup with options to check weather, tides and the status of the fishery.

Weather and Tide Display: Weather and tide data is fetched using Weather-Underground JSON API and displayed, along with a graph of tide data.

Status/Sample Update Interface: The user is able to record data, such as status of fisheries and samples. If there is no internet connection, data is stored in local storage and sent to the server when the a connection is available.
